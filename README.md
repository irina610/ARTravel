🌍 ARTravel - Cluj-City Tour Planner
ARTravel este o aplicație web dezvoltată în Django care ajută utilizatorii să exploreze orașul Cluj-Napoca. Aplicația permite vizualizarea atracțiilor turistice, a restaurantelor și a punctelor de interes medical, oferind în același timp integrare cu Google Maps pentru planificarea traseelor.

✨ Funcționalități Principale
Explorare Locații: Vizualizarea unei liste detaliate de atracții turistice, restaurante și farmacii/spitale din Cluj-Napoca.

Sistem de Favorite: Utilizatorii autentificați pot salva locațiile preferate pentru o consultare ulterioară.

Generator de Traseu: Generarea unui traseu optimizat direct pe Google Maps între mai multe locații selectate de utilizator.

Interfață Intuitivă: Design realizat cu HTML/CSS, adaptat pentru o navigare ușoară.

Bază de Date Dinamică: Gestiunea locațiilor și a utilizatorilor prin intermediul ORM-ului Django.

🛠️ Tehnologii Utilizate
Backend: Python 3.x, Django Framework

Frontend: HTML5, CSS3, JavaScript

Bază de date: SQLite (dezvoltare) / PostgreSQL (producție)

API-uri: Google Maps JavaScript API / Directions API
